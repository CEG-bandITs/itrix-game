# itrix-game

Game Website devoloped for ITRIX-2022.

# Building And Running the Project

Clone the repo

```sh
git clone https://github.com/CEG-bandITs/itrix-game.git
```

Installing Dependancies

```sh
npm install
```

Starting the project

```sh
npm run start
```
